# 沉浸式修仙云存档后端

这是一个为《沉浸式修仙》面板定制的云存档后端服务，可以让你的游戏存档在云端同步。

## ✨ 特性

- 存档、读档、删除、列出存档
- 兼容 Zeabur 一键部署

## 🚀 一键部署到 Zeabur

点击下面的按钮，Fork 本仓库到你自己的 GitHub 账户，然后授权 Zeabur 即可自动完成部署！

[![Deploy to Zeabur](https://zeabur.com/button.svg)](https://zeabur.com/templates/new?template=https://github.com/你的GitHub用户名/你的仓库名)

## 🔧 使用方法

1.  **部署服务**：点击上面的 "Deploy to Zeabur" 按钮，按照提示完成部署。
2.  **获取 API 地址**：部署成功后，在 Zeabur 控制台找到你的服务，点击域名，复制完整的 `https` 地址。例如 `https://your-service-name-xxxx.zeabur.app`。
3.  **配置面板**：将获取到的 API 地址粘贴到《沉浸式修仙》面板的云存档设置中的 "服务器地址 (ApiUrl)" 输入框内。
4.  **开始使用**：享受云存档带来的便利吧！
